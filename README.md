## Variables

- `API_HASH` Your API HASH Get From my.telegram.org
- `API_ID` Your API ID From my.telegram.org
- `BOT_TOKEN` Your Bot TOKEN Get From @BotFather
- `UPDATE_CHANNEL` Channel Username Without @
